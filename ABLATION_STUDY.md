# Ablation Study

The ablation study identifies which OCR stages reduce CER. Every single-variant
preprocessing row uses the same single-pass PaddleOCR baseline configuration, so
OCR parameter changes are not confused with preprocessing contributions.

This is a module-level ablation. The single preprocessing rows independently
replace the raw image with one variant and are not cumulative with each other.
The `blue_ink` and `line_removed` images include their upstream rectification,
illumination normalization, and sharpening steps. The ensemble and LLM rows use
the normal pipeline outputs.

| Setting | CER ↓ | 說明 |
|---|---:|---|
| Raw image + PaddleOCR | generated after evaluation | baseline |
| + document rectification | generated after evaluation | 只加文件矯正，使用單一 document variant |
| + blue ink extraction | generated after evaluation | 看藍筆萃取是否有效 |
| + line removal | generated after evaluation | 看去橫線是否有效 |
| + multi-variant ensemble | generated after evaluation | 看 ensemble 是否有效 |
| + rule-based normalization | generated after evaluation | 只套用 deterministic normalization |
| + LLM correction | generated after evaluation | 在 rule-based normalization 後加入 grounded LLM |

Run the full experiment:

```powershell
.\env\Scripts\python.exe main_pipeline.py `
  --input dataset\OCR_test_lin.jpg `
  --evaluate
```

Outputs are written under `output/04_evaluation/<stem>/`:

- `ablation_report.md`
- `ablation_report.json`
- `ablation_report.csv`

The report includes strict CER, whitespace-insensitive CER, character accuracy,
and CER change relative to the raw baseline. A negative CER change means the
setting improved recognition.

`clean_document` (red-annotation suppression) and `binary` are generated by
preprocessing but are not active OCR sources in the current default ensemble.
They are therefore excluded from the main contribution table; they cannot explain
the current ensemble result.

The final two rows isolate the correction-stage contribution. Their parent
comparison is `ensemble -> rule-based normalization -> grounded LLM`, so the final
row reports the incremental LLM effect after deterministic normalization.
